# shopping-backend
A backend for a shopping website

[Frontend project](https://github.com/madsvnielsen/shopping-frontend)

# Tech stack
* node.js 🟩
* Express 🌐
* Sequelize 🧩
* mySql db 🎁
* Apache web server



## How to run
Ensure you have all dependencies installed with.
`` bash
npm install
``

For development
``bash
npm run dev
``
Build to JavaScript
``bash
npm run build
``
